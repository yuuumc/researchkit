# v1.0 — OKX AI Genesis Hackathon Submission

ResearchKit OS — AI Research Operating System. Multi-agent pipeline that turns any paper into a structured knowledge card.

---

## 🎯 What's in v1.0

### Architecture
- **6-agent pipeline**: Planner → Reader + Analyzer + Terminology (parallel) → KnowledgeBuilder → Recommendation → Export
- **Reflection + Replan loop**: self-correction when quality is below threshold
- **Adaptive prompting**: Planner LLM dynamically decides which agents to invoke based on input type
- **Locale-aware two-stage reasoning**: source-language understanding + target-language rendering (zh-CN / en-US / ja-JP / ko-KR / fr-FR / de-DE / es-ES)

### Knowledge Card output
- 🎯 Takeaway / 💭 Why It Matters / ✨ What Surprised Me / 👥 Who Should Read
- 💡 Innovation / 🔧 Methodology / 🧪 Experiments / 📊 Results / ⚠️ Limitations
- 🔮 Future Work / 🚀 Applications / 📁 Datasets
- 🔤 Terminology DAG (with `importance` + `prerequisite` fields)
- 📚 4-intent Recommendations (`improve` / `challenge` / `apply` / `survey`)
- Quality scoring (completeness / confidence / evidence)

### Inputs
- Plain text
- URL (with SSRF protection)
- PDF upload (via `pdf-parse` v2)
- Batch URL processing (configurable concurrency)

### Exports
- Markdown (GitHub-flavored)
- Obsidian (with bi-directional links)
- Knowledge Graph (Mermaid `mindmap` with escape hardening)

### UI
- Accordion + Simple/Advanced dual-mode
- 15+ micro-animations (hover lift, click ripple, staggered entry, breathing logo, glow pulse)
- Real-time SSE progress panel synchronized with backend execution stages
- First-screen-priority layout (input box visible without scrolling)
