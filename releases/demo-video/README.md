# Release Demo Videos

This directory stores demo video files referenced in GitHub Releases and the X participation post.

## Naming Convention

```
v<version>-demo.mp4

# Examples:
v1.0-demo.mp4
v1.1-demo.mp4
v2.0-demo.mp4
```

## Constraints

- **Duration**: ≤ 90 seconds (OKX AI Genesis Hackathon rule)
- **Format**: MP4 (H.264 video + AAC audio)
- **Resolution**: 1920×1080 (1080p) or 1280×720 (720p)
- **File size**: ≤ 100 MB (Twitter upload limit)
- **Audio**: clear English voiceover (or subtitles)

## Recording Checklist

Before recording:
- [ ] Close all other apps to free RAM
- [ ] Set display resolution to 1920×1080
- [ ] Prepare the demo script (see `../../demo-script.md`)
- [ ] Open browser at https://researchkit-mu.vercel.app
- [ ] Pre-load the example (click "Load Example")
- [ ] Test screen recording audio levels

After recording:
- [ ] Trim to ≤ 90 seconds
- [ ] Export as MP4 H.264
- [ ] Compress if > 100 MB (Handbrake, preset "Fast 1080p30")
- [ ] Upload to Twitter as video attachment
- [ ] Save copy here with version-tagged filename
