## Day 1 — Critical + 5 High

`commit 9b50211`

### C1 Critical — API Key 从 cookie 移除
- apiKey 改存 localStorage（不写 cookie），cookie 只存非敏感字段（type/baseURL/model）+ HttpOnly
- server-provider.ts 读 cookie 时 apiKey 从 `process.env.OPENAI_API_KEY` 补全

### H1 — /api/tools/call 加 rate limit + 工具白名单
- rate limit: 20/min
- 公开仅 `web_search` + `arxiv`，`filesystem`/`memory` 需 `x-internal-key` header
- 修复 M5 bug: `Object.keys({})` → `listTools().map(t => t.name)`

### H2 — SSRF 校验
- `/api/settings/test-provider` 校验 baseURL hostname
- 拒绝 localhost / 10.x / 172.16-31.x / 192.168.x / 169.254.x（云元数据）/ IPv6 private

### H3 — fetch-url rate limit
- `/api/research/fetch-url` 加 rate limit (15/min)

### H4 — 移除生产环境 stack trace
- `multi-agent` + `multi-agent-stream` 在 `NODE_ENV === 'production'` 时不返回 `debug.stack`

### H5 — pluginId 格式校验
- `/api/plugins/install` 校验 `^[a-z0-9-]{1,64}$`
